# **Projet tutoré n°18 - Localisation d'un robot dans un environnement discret** #

***

Ce projet a pour but d'expliquer simplement  les problematiques de localisation et de cheminement  dans un environnement discret (ici, un labyrinthe) a des etudiants debutants lors de salons ou conferences.

***

## **Comment bien commencer avec le projet**

Pour exécuter le projet, il y a trois manières de faire :

-	Via **Java** : java -jar code/jar/Maze.jar
-   Via **Eclipse** : il faut exécuter la **classe Fenetre**.
-   Via **Ant** : aller dans le dossier ```Code``` et exécuter la commande **ant** ou **ant run**,


***

## **Notice d'utilisation**

Pour utiliser correctement le projet, une notice d'utilisation a été faite : `notice.pdf`

## **Construit avec**

* [Ant](http://ant.apache.org/) - Ant pour exécuter, compiler, lancer les tests et générer un **jar**.

## **Versions**

* Version 1 :
	* Création de labyrinthe
		* Création de labyrinthe en dessin
		* Déplacement du labyrinthe
		* Choix de la taille
		* Sauvegarde et ouverture de labyrinthe
	* Algorithme de localisation
* Version 2 :
	* Algorithmes locaux :
		* Main droite
		* Pledge
		* Tremaux
	* Algorithmes globaux :
		* A*
		* Dijkstra
	* Système de player pour la gestion des algorithmes
	* Panneaux unifiés pour chaque algorithmes
* Version 3 :
	* Moteur 3D et vue associée
	* Optimisation des algorithmes
	* Génération du fichier jar
	* Utilisation de Ant
